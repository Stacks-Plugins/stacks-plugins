# Tool overview

Every framework plugin exposes the same **19 tools** defined in `@stacks/agent-core`'s `STACKS_TOOLS` registry.

| `stacks_get_balance` | Read | [Account](/tools/account) | STX, fungible, and NFT balances |

| `stacks_get_account_history` | Read | [Account](/tools/account) | Paginated transaction history |

## Read vs write

<CardGroup cols={2}>
  <Card title="Read tools (11)" icon="magnifying-glass">
    Query chain state via Hiro API or external quote APIs. No private key required.
  </Card>
  <Card title="Write tools (8)" icon="pen-to-square">
    Sign and broadcast transactions. Require `senderKey` (or `STACKS_SENDER_KEY` for Hermes).
  </Card>
</CardGroup>

## Common parameters

All tools accept:

<ParamField body="network" type="string">
  `mainnet` or `testnet`. Defaults to `mainnet`.
</ParamField>

Write tools additionally accept:

<ParamField body="senderKey" type="string" required>
  Hex-encoded private key for the signing wallet.
</ParamField>

<ParamField body="fee" type="string | number">
  Transaction fee in microSTX. Auto-estimated when omitted.
</ParamField>

<ParamField body="nonce" type="string | number">
  Account nonce. Auto-fetched when omitted.
</ParamField>

## Response shapes

**Read tools** return typed JSON objects (see category pages for fields).

**Write tools** return a broadcast result:

```json
{
  "txid": "0x...",
  "success": true
}
```

On failure:

```json
{
  "txid": "",
  "success": false,
  "error": "Not enough funds",
  "reason": "..."
}
```

## Amounts

Unless noted otherwise, STX amounts are in **microSTX** (1 STX = 1,000,000 microSTX).

Fungible token amounts use the token's smallest unit as defined by its contract.
