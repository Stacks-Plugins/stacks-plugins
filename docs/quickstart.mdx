# Quick start

Get the Stacks plugins running locally in a few minutes.

## Prerequisites

- Node.js 22+
- npm
- Python 3.9+ (only if using the Hermes plugin)

## 1. Clone the repository

```bash
git clone https://github.com/Stacks-Plugins/stacks-plugins.git
cd stacks-plugins
```

Each package under `plugins/stacks/` is standalone — there is no root `package.json` or npm workspace.

## 2. Build packages

Framework adapters depend on **`@sugarhi11/agent-core`** from npm (installed via each package's `npm install`). To develop agent-core locally, build it first:

```bash
cd plugins/stacks/agent-core
npm install
npm run build
```

Then build the framework plugin you need:

```bash
# ElizaOS
cd ../eliza
npm install
npm run build

# OpenClaw
cd ../openclaw
npm install
npm run build

# Hermes (Node bridge only — no TypeScript build)
cd ../hermes
npm install
```

## 3. Configure environment

Copy the example env file or export variables in your shell:

```bash
cp plugins/stacks/.env.example plugins/stacks/.env
```

Edit `plugins/stacks/.env` (or set in `~/.hermes/.env` for Hermes):

```bash
STACKS_NETWORK=testnet
STACKS_SENDER_KEY=your_testnet_private_key_here
# STACKS_WALLET_ADDRESS=ST...   # optional; derived from sender key on active network
# BITCOIN_PRIVATE_KEY=...       # sBTC peg-in only
# BITCOIN_ADDRESS=tb1...        # sBTC peg-in only
```

See [Configuration](/configuration) for all environment variables.

Optional smoke scripts under `scripts/sbtc-zest-smoke/` verify read-only tools, guard-rails, and adapter parity without an agent framework.

## 4. Run tests

OpenClaw includes Vitest unit tests:

```bash
cd plugins/stacks/openclaw
npm test
```

Agent-core ships with a Jest placeholder (`npm test` in `plugins/stacks/agent-core`).

## 5. Install a framework plugin

<Steps>
  <Step title="ElizaOS">
    Import and register the plugin in your agent:

    ```typescript
    import { stacksPlugin } from 'plugin-eliza-stacks';

    const agent = {
      plugins: [stacksPlugin],
      // ...
    };
    ```

    [ElizaOS guide →](/frameworks/eliza)
  </Step>
  <Step title="OpenClaw">
    Install from the local path:

    ```bash
    openclaw plugins install ./plugins/stacks/openclaw
    ```

    [OpenClaw guide →](/frameworks/openclaw)
  </Step>
  <Step title="Hermes">
    Symlink or copy the plugin into your Hermes plugins directory:

    ```bash
    ln -s "$(pwd)/plugins/stacks/hermes" ~/.hermes/plugins/stacks
    ```

    Enable it in your Hermes config and set `STACKS_NETWORK` / `STACKS_SENDER_KEY` in `~/.hermes/.env`.

    [Hermes guide →](/frameworks/hermes)
  </Step>
</Steps>

## Preview docs locally

To preview this documentation site with Mintlify:

```bash
npm i -g mintlify
cd docs
mintlify dev
```

Open `http://localhost:3000` in your browser.

<Warning>
  Never commit `.env` files or private keys. Use testnet keys for local development only.
</Warning>
